Welcome to FatBeacon!

A FatBeacon is an experimental beacon type for offline web pages. A FatBeacon does two things:

  • It advertises the name of the device
  • Runs a GATT service that offers up a simple web page.

When you select a FatBeacon from the list of nearby beacons, the app connects to it and downloads to page to your phone before displaying it. This means you can interact with a FatBeacon with no data connectivity at all.
